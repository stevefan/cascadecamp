# Cascade Camp Field Book

This folder is a small print-and-web sidecar for camp operations material.

## Public version

`public/index.html` is safe to publish and can live at:

```text
https://cascade.camp/field-book/public/
```

It should only include public-friendly information: what to pack, meal rhythm,
site norms, session culture, lightweight maps, and playful prompts.

## Private version

`private/` is ignored by git. Use it for local drafts that include privileged
information: exact address, emergency contacts, staff phone numbers, first-aid
lead, safety number, vendor details, and internal organizer notes.

Do not commit private field book files to this public repo.

## Printing

Open `field-book/public/index.html` in a browser and print:

- Destination: printer or Save as PDF
- Paper: Letter
- Layout: Portrait
- Margins: Default or minimum
- Background graphics: On
- Scale: 100%

For booklet-style printing, save to PDF first, then print the PDF double-sided
with short-edge binding if your printer supports it.
